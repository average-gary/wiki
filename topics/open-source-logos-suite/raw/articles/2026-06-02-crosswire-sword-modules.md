CrossWire's canonical module listing for the SWORD project (the OSS Bible-software backend used by AndBible, Xiphos, BibleTime, PocketSword, and many others). As of 2026-06-02, the modern English Bible offerings are public-domain or open-licensed: Berean Standard Bible (BSB), International Standard Version (ISV), Lexham English Bible (LEB), GOD'S WORD to the Nations, World English Bible, New Heart English Bible (and variants). Notably absent: ESV, NIV, NASB, NLT, CSB. The publishers of those translations have either withdrawn or never granted SWORD distribution rights. This shapes the floor for any OSS Bible app: if you are downstream of SWORD, your default modern-English options are BSB / WEB / LEB. Anything else requires the user to hold the license themselves (BYO API key, or sideload a personally-acquired module).
