# TIGERSearch

TIGERSearch is a search engine for treebanks built around the TIGER-XML graph format. The query language uses attribute-value matrices and relational operators on graph nodes (similar in spirit to Tregex but on labelled directed graphs rather than strict trees). It is GPL but explicitly unmaintained: the official page now reads "TIGERSearch is not updated anymore. You may be interested in our new tool, ICARUS." For Bible-study syntactic search the historical interest is that Logos's Cascadia datasets and similar projects often used a TIGER-XML interchange format, and the TIGER query syntax was the prior art for graph-shape pattern queries. The successor ICARUS is itself little-used and Java-based. christ-is-lord cannot reasonably embed TIGERSearch and should treat its query syntax only as a comparison point alongside Tregex.
