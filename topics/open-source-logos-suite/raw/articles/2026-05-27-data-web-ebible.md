# World English Bible (WEB)

## Key findings
- **Public domain** — no attribution required, no royalties, no API rate limits.
- Available editions: classic WEB (uses "Yahweh"), WEBP ("LORD"/"GOD"), WEBBE (British/International spelling), WEB Catholic (with deuterocanon).
- Formats: HTML, USFM (Unified Standard Format Markers — the publishing-grade XML for Bibles), EPUB3, Sword modules, PDF, MP3/OGG audio, downloadable ZIP.
- eBible.org is the canonical distribution hub — Michael Johnson maintains hundreds of public-domain and CC-licensed translations.
- Companion: **WEB-Strongs** edition is also available (WEB text linked to Strong's numbers).

## Notable quotes / specifics
- Footer marks WEB as "Public Domain" with corresponding copyright page.
- eBible.org also hosts other PD or CC-BY translations in dozens of languages — the de facto "open Bible CDN."

## Source notes
- For a free product, default English choice. Quality is "decent modern English"; not as polished as ESV/NIV but readable and accurate.
- USFM format is the right pipeline input — convert to your internal model once, render to whatever.
- Audio Bibles via eBible.org are typically also PD (Faith Comes By Hearing has parallel free-license offerings, but check per-language).
