Tyndale House's NLT API, still live as of 2026-06-02. Two-tier access: anonymous (no key, capped at 50 verses/request and 500 requests/day) and key-based (500 verses/request, 5,000 requests/day). Terms require non-commercial use unless separately licensed and an affirmation that "your use is consistent with Tyndale's purpose." The page does not include an explicit revocation/termination clause comparable to Crossway's "for any reason" language, nor does it spell out a doctrinal-conformity clause as Crossway does. The non-commercial requirement still excludes ad-supported and donation-supported apps. Documentation is under api.nlt.to/Documentation. Key passed as `key` query parameter (less secure than Crossway's header approach but unchanged from earlier years).
