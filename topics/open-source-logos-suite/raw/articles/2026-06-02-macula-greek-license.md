# MACULA Greek licensing

The MACULA Greek Linguistic Datasets are licensed CC BY 4.0 with attribution string "MACULA Greek Linguistic Datasets, available at https://github.com/Clear-Bible/macula-greek/". Per-source statements: Nestle1904 retains its upstream terms; SBLGNT references Logos Bible Software's repo for license terms (the SBL Greek New Testament is freely usable for non-commercial study with notice); MARBLE word-sense data is included with permission from United Bible Societies; Berean Study Bible glosses entered the public domain on 2023-04-30; Cherith glosses are CC BY 4.0. Critically, there is no ShareAlike clause on MACULA Greek itself, so derivative indexes (SQLite tables, Tantivy postings, JSON tree-pattern caches) can be redistributed under any compatible license as long as attribution is preserved. This is a different licensing posture from the older biblicalhumanities/greek-new-testament lowfat repo (Robie/Palmer 2014-2017, CC BY-SA 3.0), which would impose ShareAlike — christ-is-lord should ingest from Clear-Bible/macula-greek to keep license obligations limited to attribution.
