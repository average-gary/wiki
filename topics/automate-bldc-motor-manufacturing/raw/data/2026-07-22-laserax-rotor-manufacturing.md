# Rotor Manufacturing Process (Laserax)

**6-step rotor chain:** Design/prototype → Lamination stacking → Magnet/conductor insertion → Machining + laser marking → Shaft cleaning + press-fit → Impregnation + balancing.

- **Lamination = fully automated**: hundreds of thin silicon-steel slices stamped/stacked on hydraulic presses; each slice gets a micron-level corrosion coating; laminations cut eddy currents + add strength.
- **Magnet/conductor insertion = manual-to-semi-automated (a bottleneck)**: IPM (interior permanent magnet) most common in EVs — magnets embedded inside the laminated core (higher RPM, thermal isolation); carbon-fiber wrapping for high-performance rotors to resist centrifugal stress.
- **Shaft press-fit = semi-automated**: shaft laser-cleaned (removes oil/dust) + laser-marked, then rotor core pre-heated and shaft pneumatically press-fit for an interference fit.
- **Balancing = fully automated, hard-tolerance**: rotors spin 10,000+ RPM; accelerometers, tachymeters, scales measure mass distribution and iterate corrections until vibration spec met.
- Laser data-matrix/serial marking on the lamination stack enables full traceability.
