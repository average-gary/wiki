# Equipment & Turnkey-Line Vendor Map

## Winding machines (needle / flyer / linear / hairpin)
- **Marsilli** (Italy, Castelleone/Cremona) — 80+ yr; spindle + flyer winding, single to 24 spindles, wire 0.02–2.5 mm, to 25,000 RPM; **needle winding on open + closed stators**. Turnkey lines, stand-alone machines, pre-owned.
- **Aumann AG** (Germany, Beelen; + China, USA) — e-mobility special machines. Named models: **A-LWS** (linear), **A-NWS** (needle), **A-FWS** (flyer); also **hairpin** stator + turnkey lines.
- **Elmotec-Statomat** (Germany, owned by **Schaeffler**) — needle, flyer, twist-free, **wave/continuous-hairpin**, linear winding.
- **GROB-Werke** (Germany, Mindelheim) — large-series stators/rotors; round + rectangular (**hairpin**) wire; four winding techs (Hairpin / Continuous Hairpin / Insertion-flyer / Needle for rotors).
- **Odawara Automation** (Japan) — needle + high-speed stator winding + hairpin.
- **Nittoku** (Japan) — FA/coil-winding machines, multi-axis mass-production to tabletop; GWM nozzle-guide inner winders.
- **China tier** (small-motor mass production): HONEST Automation, BSTP/Bestop, DETZO (FDM-2002M), IMA Automation, Huaguanie, NTO Machinery.

## Turnkey line integrators / assembly automation
- **Comau** (Italy, Turin) — E-Transmissions & Hairpin Assembly systems, Racer-5 robots.
- **Marsilli / Aumann / GROB** — also full turnkey lines.
- **Krause Automation** (Germany) — the divested **thyssenkrupp System Engineering** e-mobility unit (historically hairpin stator lines). *Update stale thyssenkrupp references.*
- **China turnkey**: Ningbo Nide Tech (induction/BLDC/universal), MNY Intelligent Technology (hairpin/e-drive, NEV/robotics/eVTOL).
- **Robotics/handling in cells**: KUKA, ABB, FANUC, ATS Automation, Manz.

## Magnetizers & magnet handling
- **Laboratorio Elettrofisico** (Italy, Nerviano) — magnetizing systems + measurement integrated with automation; iMag Master series.
- **Magnet-Physik Dr. Steingroever** (Germany, Cologne) — magnetizers, custom fixtures/coils, handling; PERMAGRAPH hysteresigraph, fluxmeters, field scanners; ISO 17025 lab.
- **Brockhaus Measurements** (Germany) — magnetic material / lamination testing.
- (Equipment side also: Rimac Machines, MagnetAct, Magnetic Instrumentation — see magnet-bonding data note.)

## Lamination stamping / stacking presses
- **Schuler Group** (Germany, Göppingen; now part of **ANDRITZ**, Austria) — cutting systems, high-speed presses, notching lines; E-Mobility division.
- **Bruderer AG** (Switzerland) — high-speed stamping presses; B3 control includes integrated **BSP lamination-stacking** control.
- **Kienle + Spiess** (Germany) — lamination stacks/rotors/stators as a component supplier; **Oberg** (USA) — progressive dies/tooling.

## End-of-line test
- **Schleich GmbH** (Germany, Hemer) — surge (PD), IR, hi-pot to **6 kV AC / 8500 V DC**, winding resistance, PI/DAR, function test. Products: **MTC3 R2**, **MotorAnalyzer3**, GLP series, scalable Matrix (6–24 test points).
- **Marposs** (Italy) — E.D.C. EOL systems (see EOL testing data note).
