# Machining Ti-6Al-4V — Tool Wear & Coolant

- **Low thermal conductivity ~6.7 W/m·K** → **~80% of cutting heat is trapped at the cutting edge** (vs carried away in chips for steel/aluminum). Root cause of accelerated tool wear.
- **Work hardening:** titanium hardens instantly under pressure; if the tool rubs rather than shears, the subsurface hardens and destroys the next pass → need constant feed + sharp edges.
- **Chemical reactivity:** at high temperature titanium welds to the tool → built-up edge (BUE) and catastrophic failure.
- **Parameters:** conservative **surface speed 150–200 SFM (45–60 m/min)**; moderate/constant feed so the tool always bites fresh material. Tooling: high-performance carbide with **PVD TiAlN coatings** (thermal barrier); fewer robust flutes for roughing (chip evacuation), high flute count for finishing.
- **Coolant:** **high-pressure coolant (HPC) extends tool life 30–60%** vs flood; **cryogenic cooling up to 4.5× tool life** vs emulsion flood.

**Automation implication:** to run Ti machining unattended, cells need **tool-wear/vibration monitoring, redundant "sister" tooling** (why medical FMS carry 330–530-tool magazines), and aggressive HPC/cryogenic coolant. This is the physics behind the "lights-out titanium is hard" reality.
