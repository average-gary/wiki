# Flexxbotics + Mach Medical — Lights-Out Ortho Cell

**The single most concrete "maximal automation" datapoint.** Independently surfaced by the CNC, metrology, and vendor research agents (triangulated → high confidence).

- **Fully autonomous multi-machine workcell running 24/7 lights-out**: 5-axis vertical milling + **in-line CMM** + custom part wash/blow-off, all orchestrated by a **single robot**. (Modern Machine Shop names the stack: **Okuma 5-axis + Universal Robots cobot + CMM + Flexxbotics** software.)
- **Autonomous closed-loop process control**: CMM inspection results are automatically routed back to the CNC machines in real time → the cell self-corrects without an operator. This is the key mechanism making lights-out feasible for *regulated* implants.
- **Flexxbotics FlexxCORE** connects/coordinates the robot with existing heterogeneous CNC equipment + IT systems ("full command and control") — an interoperability layer.
- Quantified gains: **+61% production capacity, +44% throughput, 20:1 machine-to-labor ratio, 3-week lead time (from 20 weeks), scrap <1%, lot sizes as low as one, up to 83% inventory reduction.** Parts held to **<25% of allowed 1 mm tolerance.**
- **Full FDA compliance incl. per-implant Device History File (DHF) traceability.** "High Velocity Single-Piece Flow Manufacturing." Current unattended runtime ~36 h, expandable.
- Implant types: joint-replacement (knee/hip).

**Why it matters:** demonstrates that the *machining + metrology + cleaning* core of the chain can genuinely run lights-out today, with the regulatory validation satisfied via automated closed-loop control + born-digital DHR — the concrete proof-of-concept for the whole topic.
