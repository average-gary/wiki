# Bitcoin Wiki — P2Pool

Canonical baseline for P2Pool's origin and core terminology.

## Origin
- Created by **Forrest Voight**
- Announced 2011-06-17 (bitcointalk)
- Officially founded 2011-07-17
- First public review 2011-07-26

## Core innovation
Sharechain — a parallel blockchain with 30-second difficulty, used as the consensus medium for share submissions.

## Terminology introduced
- **DOA (Dead On Arrival) shares** — shares that arrive too late to count
- **Orphan shares** — shares on the losing fork of the share-chain

These terms are still used in p2poolv2's accounting code.
