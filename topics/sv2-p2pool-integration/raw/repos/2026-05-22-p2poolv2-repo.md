# p2poolv2/p2poolv2

Canonical repository for the "Reboot P2Pool" project. License: AGPL-3.0. Active development; ~76 stars at time of ingest, >540 merged PRs, 1,644 commits.

(Some search results refer to `pool2win/p2pool-v2` — that account name appears in commits and earlier prototypes; the canonical org is `p2poolv2`.)

## Mission
> Replace pool operators with miners.

## Workspace structure (7 crates)
- `bitcoindrpc` — Bitcoin Core RPC client
- `p2poolv2_config` — config parsing
- `p2poolv2_lib` — core library
- `p2poolv2_node` — node binary
- `p2poolv2_cli` — CLI
- `p2poolv2_api` — HTTP API (web dashboard)
- `p2poolv2_tests` — integration tests

## Toolchain
- Edition 2024, MSRV Rust 1.88.

## Networking
- **libp2p 0.53** with TCP, DNS, Tokio, Noise, Yamux, Kademlia, secp256k1.
- Peer-to-peer share gossip and node discovery.

## Bitcoin layer
- `bitcoin 0.32.5`
- `bitcoinconsensus 0.106.0`
- `zmq 0.10` — direct Core integration via RPC + ZMQ (rawblock/hashblock topics).

## Storage
- `rocksdb 0.24`

## Runtime
- tokio (full features), dashmap, async-trait, tracing.

## Critical observation: NO SV2 dependencies
**No `stratum-common`, `binary_sv2`, `roles_logic_sv2`, `stratum-core`, or sv2-apps crates anywhere** in the workspace. p2poolv2 is V1-only today. SV2 integration is greenfield.
