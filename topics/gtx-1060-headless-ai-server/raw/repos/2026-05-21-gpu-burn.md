# gpu-burn

## Build for Pascal (CRITICAL)

```bash
git clone https://github.com/wilicc/gpu-burn
cd gpu-burn
make COMPUTE=6.1                       # critical for GTX 1060
```

> Default `make` targets newer compute capabilities → silent misbehavior on sm_61.

## Smoke + burn-in commands

```bash
./gpu_burn 600                         # 10-min smoke test
./gpu_burn -d 3600                     # 1-hour double-precision burn-in
./gpu_burn -m 5500 1800                # cap at 5.5 GB (leave headroom on 6 GB card), 30 min
```

## Why

Exercises both compute and memory. If gpu-burn passes a 1-hour double-precision run AND temperatures stay safe, the GPU + driver + power delivery are healthy enough for sustained inference workloads.

## Companion CPU/RAM tests

```bash
stress-ng --cpu $(nproc) --vm 2 --vm-bytes 75% --timeout 600s --metrics-brief
sysbench cpu --threads=$(nproc) --time=60 run
sysbench memory --memory-total-size=10G run
```
