# FCSO + Winchester + NRADC — Staffing and State-Funded Budget

Primary-source data from the **Virginia Compensation Board** (SCB), the agency that funds VA constitutional officers (sheriffs, treasurers, etc.) at the state share. Local supplements (the bulk of FCSO's actual budget) live on `fcva.us` adopted-budget PDFs which remain blocked.

## Critical structural fact: FrCo operates NO county jail

Per the Compensation Board FY19 Staffing Standards PDF, Frederick County row (FIPS 069) shows **0 corrections beds, 0 ADP, 0 corrections staff**. Same for Winchester (FIPS 840). All jail function is handled by the **Northwestern Regional Adult Detention Center (NRADC)**, FIPS 410.

## NRADC (Northwestern Regional Adult Detention Center)

| Metric | FY19 |
|---|---|
| Population catchment | ~197,235 (FrCo + Clarke + Winchester) |
| **Operating capacity** | **556 beds** |
| **Average daily population** | **627** |
| **% above capacity** | **+13%** |
| Sworn corrections positions HAVE | 134 |
| Positions DUE (at 1:3 + overcrowding factor) | 199.55 |
| Additional positions DUE | 65.55 |
| **% of state-defined need** | **10.61%** |

NRADC is **operating ~13% above capacity with only 11% of staffing need met.**

### Staffing density

State-funded only: **53 deputies / 96k population = 0.55 per 1,000.**

This excludes locality supplements. Real FCSO total is higher — the county pays out of the General Fund for additional positions, equipment, fleet, training. Without the FrCo adopted budget (blocked), the supplement size is unknown.

State methodology says FCSO is owed +13.83 more positions to meet statewide standard.

## Winchester Sheriff (independent city, court-services only)

Winchester is an independent city. Its **Sheriff's Office handles only courthouse security and civil process service** — NOT primary law enforcement (that's Winchester PD, a city-funded municipal agency, NOT in CB data).

### FY19 staffing

| Metric | Value |
|---|---|
| 2017 population basis | 28,005 |
| State-funded LE deputies | **0** (city PD does LE) |
| State-funded Court Services | **5.0** |
| Additional CS due | 2.20 |
| **% of state-defined need** | **43.95% (3rd worst statewide)** |

### Court services workload (CY17)

- Circuit days: 155
- General District days: 386
- J&DR days: 485
- Papers received (3-yr avg): 15,969
- Total court-service hours: 12,596

### FY24 Compensation Board salary base

| Line | Amount |
|---|---|
| Current salary base (Sept 2023) | **$434,241** |
| Sheriff consolidation of population groups | $10,460 |
| Targeted initiatives total | $12,360 |
| 2% across-the-board | $8,932 |
| **Estimated salary base 12/1/2023** | **$455,533** |

The size disparity vs FCSO ($3.3M vs $0.43M) reflects the structural difference: FCSO does primary patrol for ~96k pop in unincorporated FrCo, while Winchester Sheriff handles only court services for ~28k pop.
