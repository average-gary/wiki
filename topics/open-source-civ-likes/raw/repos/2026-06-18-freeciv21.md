# Freeciv21 — longturn/freeciv21

Modern Qt6 fork of Freeciv, maintained by the **Longturn.net** team for
competitive multiplayer. Project's own framing: *"roots in the well-known
FOSS game Freeciv and extends it for more fun, with a revived focus on
competitive multiplayer environments."* The word "revived" implicitly
concedes upstream Freeciv had let competitive multiplayer atrophy.

## Stack

- **C++ 86.6%, C 6.6%, Python 4.1%, Lua 1.5%**
- License **GPL-3.0+**
- Build: **CMake 3.21+ with Ninja** (`cmake . -B build -G Ninja`)
- Requires **Qt 6.6+** (base + SVG)

## Features

- Both **hex and square tiles**
- **>500 selectable nations**
- Turn-based 4X starting 4000 BCE

## Activity (as of 2026-06-18)

- **30,641 commits, 42 total releases**
- **v3.1.0** shipped 2024-08-10 — culmination of a two-year cycle:
  redesigned city dialog, new Units View, snap/flatpak packaging.
- **v3.1.1** (March 2025/2026 — date discrepancy in sources, see note below)

- **v3.2-dev.2** (2024-08-20) — active dev branch, Qt6 migration, Markdown
  help text, civ2 ruleset compatibility improvements, expanded Map Editor.

Distributed via Snap, Flathub, AUR, .deb, Windows installer, macOS installer.
