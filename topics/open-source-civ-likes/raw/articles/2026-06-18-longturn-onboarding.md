# How to join a Longturn game

Community-adjacent practitioner blog. The only step-by-step "join" walkthrough
for the Longturn community that runs Freeciv21 multiplayer.

## Format framing

> *"multiplayer one-turn-per-day playstyle and community... probably the
> closest thing Civilization and Freeciv will ever get to Massively
> multiplayer online strategy."*

## Three-step onboarding

1. Install the native Freeciv21 client from Freeciv.org / longturn.net
2. Register at **Longturn.net**
3. Sign up when registrations open AND **confirm participation days
   before start** — both required, missing either drops you

## Version flexibility

Usually newest server/client but not always. Older client versions are kept
available; per-game version requirement is listed on each game's page.

## Community gates

- `forum.longturn.net`
- Discord: `discord.gg/98krqGm`

## Cadence

**One-turn-per-day** is the only deadline mechanic surfaced. No detail on
TC (turn-change) timing or per-turn extensions in this source.

## Why this matters

Once a reader knows what Freeciv21 / Longturn is, this page tells them
exactly how to get into a real game with the actual community gates. Closes
the practitioner loop.
