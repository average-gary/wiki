# Wikipedia — 0 A.D. (video game)

Encyclopedia summary covering 0 A.D.'s 16-year alpha cycle, funding, and
2026 milestone release.

## Project timeline

- **2001**: development started
- **2010**: first public alpha
- **2010–2026**: ~16 years in alpha
- **2026-02-18**: **Release 28 "Boiorix"** — the **first non-alpha
  release**, ending 16+ years of alpha designations

The 16-year alpha cycle is the canonical example of OSS 4X/RTS development
pace problems — but the project *did* eventually deliver.

## Funding constraint

**2013 Indiegogo aimed for $160,000, raised only $33,251** — insufficient to
hire substantial paid help. Project remained dependent on volunteer churn.

No formal release date for "completion" was ever set; project framed every
alpha as incremental.

## Reception

Reception in the press is broadly positive — which makes the contrarian
story more interesting. Critics aren't saying it's bad, they're saying
**it's eternal-WIP**.

## Why this matters

Anchor source for the "0 A.D. spent 16 years in alpha" claim — the single
most-cited critique of OSS 4X dev pace. Pairs with the play0ad.com primary
release announcement for the post-alpha success story.
