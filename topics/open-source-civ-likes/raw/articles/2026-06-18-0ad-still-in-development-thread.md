# Wildfire Games Forum — Is 0 A.D. still in development?

Public forum thread (Feb 2020) where the community asked whether 0 A.D.
was alive, citing a 2-year gap since the last alpha.

## Key admissions

- Project lead **Stan`** confirmed Alpha 24 was still in progress; community
  defended pace ("Rome wasn't built in a day").
- Insider **Wowgetoffyourcellphone** undercut the **"62% complete"**
  milestone metric: *"That percentage has been largely arbitrary for years"*
  — i.e. devs themselves admit progress tracking is theatre.
- Pattern: long inter-release gaps spawn "is it dead?" threads that the
  team has to answer with commit-count rebuttals (1547 commits in 14 months).

## Why this matters

Captures the project's own community asking the dead-or-alive question, with
**insider admissions about hollow milestone metrics**. Useful contrarian
counterweight to the eventual Release 28 (2026) success story — the project
*did* finish, but the path was opaque to its community for years.
