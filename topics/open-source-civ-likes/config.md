# Open Source Civ-Like Games

This wiki surveys the landscape of open-source civilization-like games:
the long-running implementations (Freeciv, C-evo), the modern revivals
(Freeciv21, Unciv), the genre-adjacent 4X projects (0 A.D., Unknown Horizons,
OpenAge), and the engines and design decisions that shape them.
