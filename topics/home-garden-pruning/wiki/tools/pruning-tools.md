# Pruning Tools

For the user's plants (cherries, blackberries, roses, raspberries, blueberries), four tools cover all cuts:

## Required tools

| Tool | Cut size | Notes |
|------|----------|-------|
| **Bypass hand pruners** | Up to ½" | Workhorse for shrubs, brambles, roses. Felco 2 / 6 / 8, Bahco PX, Corona BP — all fine. |
| **Bypass loppers** | ½ - 1.5" | Long handles for leverage. Bahco P116-SL-50, Corona ComfortGEL, Fiskars PowerGear all good. |
| **Folding pruning saw** | 1 - 4" | Silky F180 or Bahco 396-LAP. Pull-stroke saws cut on the pull, much faster than push-stroke. |
| **Long-reach pole pruner / saw** | Tall cherry limbs | Optional but useful for cherry once trees mature. Telescoping pole + saw head + bypass-pruner head. |

## What to avoid

- **Anvil pruners** for live wood — the blade-on-flat design crushes/bruises live cane. OK only for dead wood. **Never use on roses or fruit canes.**
- **Hedge shears** for shrub roses or blueberries — the indiscriminate shearing produces twiggy regrowth and inferior fruit. Use bypass pruners for selective cuts.
- **Dull tools** — sharpen blades after each session with a few strokes on a small whetstone or fine file (bevel side only).

## Sanitation supplies

| Item | Purpose |
|------|---------|
| **70% isopropyl alcohol** | Tool disinfectant (preferred for roses, cherries) |
| **Tall narrow container** | Holds alcohol for blade-dipping (tall yogurt cup, jam jar) |
| **Household bleach + spray bottle** | Backup disinfectant; mix 1:9 with water |
| **Rags / paper towels** | Wipe blades clean before sanitizing |
| **Light machine oil** (3-in-1, sewing oil) | Wipe blades after use to prevent rust |
| **Contractor bags** | Bagging diseased prunings (RRD, canker) |

## Personal protective equipment

| Item | When |
|------|------|
| **Leather gloves** | Always for roses (thorns); blackberry/raspberry (even thornless cultivars have weak prickles) |
| **Long sleeves** | Roses, brambles |
| **Eye protection** | When using a pole saw or pruning anything overhead |
| **N95 mask** | If pruning a confirmed-diseased rose (RRD), to avoid inhaling mites |

## Maintenance

After every session:
1. Wipe blades with a rag.
2. If tools touched diseased material, dip in 70% alcohol for 30 seconds.
3. Wipe blades dry.
4. Apply a thin film of oil.
5. Store in a dry place (not a damp shed).

Annually (or when dull):
- Sharpen blades with a small file or whetstone — bevel side only, follow the existing angle.
- Tighten the pivot bolt on hand pruners and loppers.
- Replace springs if tension feels weak.
- Replace blades if pitted or chipped beyond sharpening.

## Recommended kit (under $200)

For the user's 5-plant home garden:

- Felco 2 bypass pruners (~$60)
- Corona ComfortGEL bypass loppers (~$45)
- Silky F180 folding saw (~$50)
- Quart of 70% isopropyl alcohol (~$5)
- Pair of rose gloves (~$20)
- Sharpening file (~$10)

Total: ~$190. With minor maintenance, this kit lasts 10+ years.
