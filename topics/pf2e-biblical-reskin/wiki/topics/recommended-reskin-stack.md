# PF2e Biblical Reskin — recommended stack

Single-page synthesis across all 6 research paths. Each section links to the deep concept article.

## TL;DR

> **Treat the Bible as setting, not as ruleset.** Use the **PF2e Remaster** as-is (sanctification + holy/unholy/spirit/vitality/void damage are theologically literate by accident). Default to the **Catholic lens** for the richest fantasy material; **Lewisian / "mere Christianity"** for widest audience. Adopt the **hybrid Charism + Lewisian** magic-theology approach so every PF2e class stays playable while Arcane/Occult casting carries real moral stakes (Watcher angels vs fallen Watchers as the source axis). The **single biggest reskin wins** are Champion (sanctification = Beatitude-tenets engine), Thaumaturge (implements = Christian relics 1:1), Oracle (cursebound = OT prophet), and the **Nephilim ancestry** (Genesis 6 angelic-hybrid reading; the Remaster merger of Aasimar/Tiefling/Aphorite/Ganzi makes it canonical). Cut Witch / Psychic / Animist as theologically irreparable.

## How the paths connect (cross-path findings)

These are the connections the per-path agents wouldn't have found on their own:

1. **The PF2e Remaster did most of the design work for monotheism *by accident*.** Path 2 found the Remaster removed alignment, added sanctification, retyped damage to holy/unholy/spirit/vitality/void. Path 5 found that this makes Champion a Beatitude-tenets engine and Oracle an OT-prophet engine *with no rules changes*. Path 3 found that the resulting Pseudo-Dionysian-friendly celestial ladder maps cleanly onto PF2e creature taxonomy. **Conclusion**: target PF2e Remaster specifically; legacy PF2e (OGL Bestiaries 1–3) and 5e are structurally hostile by comparison.

2. **The cessationism vs continuationism axis (path 6) is the load-bearing meta-decision** that determines which magic-theology approach (path 4) is internally consistent for the chosen audience. **Reformed/cessationist → approach #3 (all-demonic) or #4 (strict-historicist).** **Catholic/Orthodox/Pentecostal/charismatic → approach #1 (charism) or hybrid #1+#2 (charism + Lewisian).** This is what determines class viability (path 5).

3. **The "DragonRaid trap" (path 1) and the "magic-theology problem" (path 4) have the same shape.** Path 1: don't make piety a mechanical input. Path 4: don't make magic literally Spirit-led prayer (or you've reproduced DragonRaid's reciting-verses-to-cast problem). The **Charism approach must be careful** — *charismata* in 1 Cor 12 are gifts not techniques, and the mechanics should reflect that: clerics petition; they don't perform.

4. **The Nephilim ancestry is the single biggest content win across all paths.** Path 3 (theology): Genesis 6 angelic-hybrid reading is second-temple consensus and gameable. Path 5 (class/ancestry): the Remaster's merger of Aasimar/Tiefling/Aphorite/Ganzi into a single Nephilim heritage makes it canonical. Path 6 (denominational): even a Protestant 66-book setting can use Nephilim because they're in canonical Genesis 6 (Enoch is supplementary, not load-bearing). **Conclusion**: the Nephilim ancestry is the headline ancestry of the reskin in every denominational lens.

5. **The Champion + Sanctification framework is the single biggest mechanical win.** Path 2: Remaster decoupled cause from alignment. Path 5: edicts/anathema attaches to deity (one entry suffices for monotheism). Path 6: works in every denominational lens — Catholic attaches saints to causes; Reformed attaches abstract virtues; Pentecostal attaches Bible-hero exemplars. **Total custom-rule footprint for Christian Champion: zero.** This is what a Christian Paladin should always have looked like.

6. **Thaumaturge's implements map 1:1 to Christian relics.** Path 5 caught this; path 3's reliquary/sacramental research confirms the depth of source material; path 6's Catholic-Recommended-Default makes it the natural class. **The Thaumaturge is the most under-appreciated design fit** — it's a pilgrim/saint-who-wields-sacred-objects with zero rules changes.

7. **Druid is the most contested class.** Path 4 (magic-theology): Primal tradition is theologically loaded — pact-with-spirits flavor is animist. Path 5 (reskin): Druid needs major reflavor (Edenic remnant, Franciscan nature-mystic) or cut. Path 6 (denominational): Reformed leans cut; Catholic/Orthodox can rehome as Franciscan. **Decision-deferred to per-setting choice.**

8. **Existing fan homebrew is invisible.** All 6 paths flagged Reddit / itch.io / DriveThruRPG / Pathfinder Infinite as hard-blocked. **Either no significant published Christian PF2e Remaster reskin exists yet, or it does and the search infrastructure can't find it.** This is itself a finding — the niche is open.

## Implementation phasing (if you actually build this as a content pack)

Cross-link with [[pf2e-worldbuilding-tool]]: this reskin would ship as a content pack for that tool. Per [[pf2e-licensing-posture]] in the sister topic, **ORC mechanics + Golarion-replaced** is the monetization-clean posture (ORC-only is monetizable; Golarion content forces Community Use which forbids paywalls; this reskin replaces Golarion entirely so it's pure ORC + homebrew).

**Phase 0 — meta-decisions (1–2 sessions of GM/designer thought)**:
- Pick a denominational lens ([[denominational-lens-decision]]).
- Pick a magic-theology approach ([[magic-theology-approaches]]).
- Pick a millennium reading + intermediate state ([[biblical-cosmology-pf2e-mapping]]).

**Phase 1 — setting bible (one writing pass)**:
- Cosmology document (planar map per [[biblical-cosmology-pf2e-mapping]]).
- One YHWH deity stat block; per-saint sub-entries if Catholic/Orthodox.
- 5 short reflavor paragraphs (one per tradition + one for Champion causes).
- 2 homebrew rules: void-trait spells = anathema for canonical clergy; harmful divine font restricted/cut.

**Phase 2 — class/ancestry reskins (lookup-table style)**:
- Per-class verdict table from [[class-ancestry-reskin-verdicts]].
- Per-ancestry verdict table.
- Champion causes as Beatitude-tenets (or saint-patrons in Catholic/Orthodox).
- Thaumaturge implement reskin (chalice / regalia / tome / lantern / etc. → Christian relic categories).
- Sorcerer bloodline cuts (Demonic, Diabolic, Hag, Shadow, Undead → cut or villain-only).

**Phase 3 — adventure scaffolding (per-campaign)**:
- Pick eschatology axis values; build endgame around restored Material Plane (or rapture-then-millennium for dispensational).
- For Orthodox lens: Aerial Toll Houses as level-15+ post-mortem mega-dungeon.
- For Catholic lens: Purgatory as transitional plane.
- For Pentecostal lens: spiritual warfare as explicit campaign frame.
- For setting-as-meta-question: Reformation / East-West Schism / Council of Nicaea as adventure-path backbone.

**Phase 4 — content-pack data (if shipping as data for the worldbuilding tool)**:
- ORC-only mechanics → monetizable ([[pf2e-licensing-posture]] in sister topic).
- Per-record license-provenance metadata.
- Foundry pf2e JournalEntry export format (matches the sister topic's recommended interop pattern).
- Cross-reference [[pf2e-remaster-name-mapping]] for any spell/feat that's been theologically renamed.
