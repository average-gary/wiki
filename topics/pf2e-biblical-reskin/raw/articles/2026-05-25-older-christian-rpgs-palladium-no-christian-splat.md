# Palladium Christian content: not surfaced

## What the gap note expected

The original brief flagged "Palladium Books published Christian-themed articles in *The Rifter* and *Heroes Unlimited* sourcebooks. Likely never their own product."

## What the search returned

**Wikipedia on Heroes Unlimited**: enumerates character classes — super-powered beings, aliens, technology-based, special-training, magic-powered, mutant animals (TMNT crossover). No Christian or religious class referenced. Game is generic comic-book superhero in tone.

**The Rifter (magazine, Wikipedia)**: 404 from this fetch — article title likely different from `The_Rifter_(magazine)`. A search of `rifterindex.dpenwood.com` (the fan-maintained article index across all 100+ issues) returned only the index page header, not searchable contents from this fetch tool.

**Bing/DDG searches**: query terms "Palladium" + "Christian" return either Palladium-the-element, Palladium Boots, or generic Christianity articles. Search engines fail at disambiguating "Palladium Books" without additional context terms. Adding `"Heroes Unlimited"` and `"The Rifter"` to the query did not surface any reference to Christian-themed articles.

## Interpretation

Two possibilities:

1. **The original claim is correct but the evidence is paywalled in PDF issues.** *The Rifter* is a quarterly fan-submission magazine (issues 1–100+ across ~25 years). A single fan-submitted article on, e.g., a "Crusader" or "Inquisitor" character class for *Heroes Unlimited* or *Rifts* would not be web-indexed. The dpenwood Rifter Index would resolve this if a deeper search tool was available.

2. **The original claim was speculative / mis-attributed.** Palladium's house style historically leaned generic fantasy and pulp-superhero rather than overtly Christian. The Rifts world includes "True Atlanteans" and various pseudo-religious factions but not specifically Christian content as a central pillar.

## Recommendation

Score this at **medium-low priority**. Even if a *Rifter* fan article on a Christian character class exists, its design influence on the modern Christian-RPG conversation is negligible — *Rifter* fan articles do not get cited in serious design retrospectives the way DragonRaid or Testament do. Worth re-checking only if `rifterindex.dpenwood.com` becomes searchable.
