# Cashu Dev Kit (CDK) v0.17.1

Rust implementation of the Cashu protocol. Released 2026-06-16.

## Stats
- **Composition**: 97% Rust.
- **Status**: alpha.
- **Backends**: SQLite, PostgreSQL, Redb, Supabase.
- **Lightning backends**: CLN, LND, **LDK Node**, LNbits.
- **crates.io**: `cdk` 36,911 total downloads, 6,017 in last 90 days; created 2024-04.

## Underlying protocol notes
- NUT-00 specifies BDHKE over secp256k1 — descriptive only, no formal cryptographic proof.
- NUT-12 adds offline DLEQ (discrete-log-equality) proofs so wallets can verify a mint signed with the same key it advertises.
