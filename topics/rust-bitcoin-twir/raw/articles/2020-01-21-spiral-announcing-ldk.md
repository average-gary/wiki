# Spiral — What We're Building: Lightning Development Kit

LDK origin announcement from Spiral (then Square Crypto, a subsidiary of Block, Inc.), 2020-01-21.

## Key facts
- **Built on top of pre-existing rust-lightning** — Matt Corallo's project, started ~2018.
- **Pivoted Lightning** from standalone-node implementations to a **modular SDK** for wallet developers.
- rust-lightning has been "in production use since 2021" per the GitHub repo.
- Funded by Spiral, which also funds rust-bitcoin, BDK, BTCPay, ZeroSync.
