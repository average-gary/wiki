# Fedimint's Ecosystem: A Look Back at 2025's First Half

Fedimint project blog, 2025-06-30. Multi-quarter review.

## Highlights
- **LDK Node integrated directly into Lightning Gateway** — collapses 2 daemons to 1 (`gatewayd`).
- **v0.7 ships beta Iroh networking**, eliminating public-IP / domain requirements for home Guardians.
- **Vipr Wallet** (PWA on Fedimint WebSDK) adds **NIP-87 mint discovery + NWC**.
- **Guardian UI now embedded** (127.0.0.1:8175) — no separate web server.
- **HRF-funded BitSacco project** running in Kenya on Fedimint.
