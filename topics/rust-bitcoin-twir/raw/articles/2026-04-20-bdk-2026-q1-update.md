# BDK 2026 Q1 Update

Quarterly recap from bitcoindevkit.org, 2026-04-20. Multiple Rust crates released in coordinated wave.

## Crates / versions
- `bdk_wallet 3.0.0` (rc1, rc2) — new migration utilities for older 0.x BDK wallets
- `bdk_chain 0.23.3`
- `bdk-ffi 2.3.0/2.3.1`
- `bdk_esplora 0.22.2`
- `bdk-cli 3.0.0` — integrated PayJoin support for testing

## Other highlights
- bdk-dart (Flutter) and bdk-rn (React Native) reached production-ready testing stage.
- Devkit Wallet UI redesigned with compact block filter (Kyoto) support.
- Formalized library support tiers and primary/secondary maintainers across the GitHub org.
