# 2024 State of Rust Survey Results

Official Rust project blog post, 2025-02-13. Already covered in TWiR around that date.

## Headline numbers
- **7,310** completed responses (out of 9,450 started; 77.4% completion).
- **38%** use Rust for majority of work coding (up from 34%).
- **45%** report org makes non-trivial use of Rust (+7pp YoY).
- **82%** say Rust helped company achieve goals.

## Top use cases
- server backends, web/networking, cloud, embedded (growing), WebAssembly (23% browser-targeted, 7% other Wasm targets).

## The Bitcoin gap
- **Zero mentions** of bitcoin, cryptocurrency, blockchain, crypto, finance, or fintech in headline categories.
- Bitcoin/crypto adoption of Rust is **invisible** in the official survey's domain breakdown.
- Yet `secp256k1` ranks among the most-downloaded crypto crates on crates.io (~14.2M/90d).
