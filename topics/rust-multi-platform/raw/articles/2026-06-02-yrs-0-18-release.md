The y-crdt monorepo's latest release is **yrs v0.18 (2025-03-12)** — the only
2025 release, and yrs remains pre-1.0. Headline changes: a **unified Observer
API** (all subscriptions reduce to a single `yrs::Subscription` type),
"logical pointer" collection references that survive across garbage
collection, and an internal-representation change aimed at memory and
efficiency. The `rand` crate was replaced with `fastrand` for **smaller
binary size and better WASM support** — a directly mobile-relevant footprint
win. Root-level XML element/text getters were removed (data-loss avoidance);
`Awareness` and the sync protocol moved to a dedicated module. Ywasm was
rewritten using logical pointers to fix segfaults. Release notes do **not**
discuss Swift/Kotlin bindings (those live in sibling repos `yswift`, `ykt`)
and do not publish concrete mobile binary-size numbers.
