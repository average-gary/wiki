# Bevy's First Birthday (2021-08-10)

## Key findings

- Week 1 reception: 3rd-most-popular r/rust post of all time, 2,200 GitHub stars in first week.
- **First funding milestone**: $1,500/month sponsorship goal hit on Day 10, anchored by **Embark Studios sponsorship** — Cart went full-time on Bevy.
- Cadence: four major releases (0.1–0.4) in first ~4 months, roughly monthly.
- **April 2021 (~0.5)**: complete ECS rewrite and PBR introduction.
- End-of-year-1 stats: 10,000 GitHub stars, 255 contributors, 4,871 Discord members.
- Confirms unified-Rust-stack design choice ("no scripting interface separating engine logic from app logic") as a year-1 ideological anchor.
