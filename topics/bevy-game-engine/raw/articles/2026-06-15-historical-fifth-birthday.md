# Bevy's Fifth Birthday (2025-08-10)

Most authoritative single-source statement on Bevy governance, funding, and 1.0 readiness.

## Key findings

- Project stats: **40,900 GitHub stars, 2.75M crates.io downloads, 1,291 contributors**, 21,985 Discord members.
- **Bevy Foundation achieved 501(c)(3) status** September 2024 — donations now tax-deductible in the US.
- Foundation funds **2 full-time employees + 1 contractor**; Carter is taking a 50%+ pay cut from market rate; Foundation is "drastically under-funded."
- 0.15 (Nov 2024): Required Components, entity picking integrated, animation graph, curves.
- 0.16 (Apr 2025): GPU-driven rendering (3x+ perf in multi-object scenes), procedural atmospheric scattering, decals, occlusion culling, ECS relationships shipped.
- **Bevy Editor still NOT shipping** despite focused work; design specs done, Feathers widget library experimental, inspector system proposed.
- **BSN (Bevy Scene Notation)**: macro-based syntax draft implementation ready for review; aim is to unify code- and asset-driven scene definition with IDE autocomplete support. *Status as of birthday post: not yet merged.*
- **Working groups model**: shifted from top-down direction to "organic" coalitions. Active groups named: GPU rendering, ray tracing, audio, CLI tooling, shader standards (WESL).
- **Notable sponsor/user**: Foresight (spatial databases/visualization). **Tiny Glade** released on Steam — flagship Bevy commercial title.
- **No explicit 1.0 timeline** — Carter avoids committing to a date; focus is "ship BSN, core UI widgets, baseline editor, reactivity."
- Forward priorities: BSN completion, reactivity ecosystem, standardized widgets, baseline editor; physics direction = Avian as upstream candidate.
