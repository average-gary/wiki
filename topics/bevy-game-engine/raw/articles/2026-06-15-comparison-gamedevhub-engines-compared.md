# Game engines compared (GameDevHub)

Broader cross-ecosystem context (not Rust-only).

## Key findings

- License/cost matrix: Godot MIT free, Unity proprietary tiered, Unreal source-available + royalties, Bevy MIT/Apache free, Phaser MIT free.
- Per-engine "best for":
  - Godot → 2D indie fast iteration
  - Unity → industry employment + 2D/3D hybrid
  - Unreal → high-fidelity 3D
  - Bevy → "learning engine internals"
- Notes Unity runtime-fee controversy and Unreal royalty thresholds explicitly.
- Default recommendation hierarchy: Godot for 2D shipping in <1yr, Unreal for AAA fidelity, Unity for hireability, Pygame/LÖVE for learning.
- Core thesis: "The engine you finish a project in is worth ten engines you research and abandon" — frames Bevy honestly as a learning/research tool today.
