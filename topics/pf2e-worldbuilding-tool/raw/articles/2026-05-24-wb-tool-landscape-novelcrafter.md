# NovelCrafter

## Pricing
"Plans starting from 4 USD/m" — 21-day free trial, no card.

## Codex (worldbuilding)
- Interactive auto-linking wiki of characters, locations, lore.
- Cross-book sharing within series — eliminates duplicated entity entry across multiple campaigns/books.
- Highlight-and-reference world elements while writing.

## AI integration — the standout feature
**Bring-your-own-key model** integrates with:
- OpenAI (GPT-5)
- Anthropic Claude
- Google Gemini
- Meta Llama
- Mistral AI
- OpenRouter (300+ models)
- **Local: LM Studio, Ollama**

User-customizable prompts with variables; "Workshop Chat" provides 24/7 partner with story/codex context.

## Other tools
- Plot planning grid + cards.
- Cross-platform incl. mobile.
- Team collaboration.

## Relevance to PF2e tool
- **Validates BYO-key LLM as a viable commercial model** — users absorb token costs; vendor avoids margin compression on inference.
- **Validates local-LLM (Ollama / LM Studio)** as a real product feature — the offline-LLM-DM pattern has precedent.
- Codex pattern (auto-linking wiki with full-context LLM access) maps almost 1:1 to a GM's prep loop: NPCs, factions, locations as entities → LLM has perfect recall when generating new scenes.
- Shows that "writers + AI" is now table-stakes; TTRPG GMs are an underserved adjacent market with the same primitives but different verbs (encounters, statblocks, loot, action economy).
