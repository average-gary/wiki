# Kanka

## Pricing tiers (2026)

| Tier | Price | Premium campaigns | File size cap |
|------|-------|-------------------|---------------|
| Kobold | Free | 0 (ads shown) | standard |
| Owlbear | $4.99/mo | 1 | 10 MiB |
| Wyvern | $9.99/mo | 3 | 25 MiB |
| Elemental | $24.99/mo | 7 | 100 MiB |

Free tier already includes unlimited campaigns and entities — paid is mostly about "premium campaigns" (more collaborators / no ads / bigger files) and roadmap voting.

## Entity model
~20 modular entity types: characters, locations, organizations, families, calendars, maps, timelines, items, journals, notes, events, races, quests, etc. Heavy use of `@mentions` for cross-linking.

## Customization
- Custom CSS allowed
- Plugin/marketplace for entity templates
- Granular permissions for collaborative campaigns

## Notable
- Open-source backend (the actual app code is public on GitHub) but hosted SaaS is the canonical experience.
- No AI features advertised.
- Active 2024–2026 development.

## Pain points (from general community knowledge — to corroborate)
- UI dense and cluttered for new GMs.
- Mobile experience is a mobile-web wrapper, not native.
- Export is JSON / per-entity but not a clean Markdown bundle.

## Relevance
Strongest free-tier competitor; sets the bar for "free with all entity types" — a paid PF2e tool must justify cost vs. Kanka free + community PF2e templates.
